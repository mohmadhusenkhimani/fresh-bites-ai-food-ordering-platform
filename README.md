# 🍔 Fresh Bites – AI-Powered Smart Food Ordering Platform

An AI-powered Full-Stack Food Ordering Platform built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

Fresh Bites delivers a modern food ordering experience with secure authentication, real-time order management, AI-powered recommendations, intelligent chatbot assistance, online payments, and a comprehensive admin dashboard.

Designed as a production-ready application, this project demonstrates full-stack software engineering principles, scalable architecture, RESTful APIs, secure authentication, and practical AI integration.
